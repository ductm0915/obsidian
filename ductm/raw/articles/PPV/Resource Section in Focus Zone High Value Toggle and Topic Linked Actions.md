## The Core Framework: Capture, Elevate, and Reuse

So the core framework for the resource section comes in three steps. Capture, elevate, and reuse.

---

## Step 1: Capture

We'll start with capture. Capture is about making actions carry context. When you first create an action, there are several context properties that are particularly useful later more than in the moment.

And these properties are category and type. So the personal work or meetings versus correspondence. And these are captured upfront but hold long-term value and will keep paying off in your system in the long run.

Another property that is very helpful if you can capture it early on is the project project tacking. So when relevant try to link to a project this action serves or is aiming for.

### Short-Term vs Long-Term Value

So all actions have short-term progress values like get mail, drop a package, make an appointment. The value is simply done versus not done to keep momentum and trigger the next step. Once done, their value decays quickly or simply ends.

But certain actions hold additional long-term value because what's inside the page, they contain useful content like notes, decisions, patterns, checklist, or other thoughts or discussions you have with other people.

If you do nothing, these actions will become the sediment in the juice box. It's loaded with nutrients and yet it sinks to the bottom and gets thrown away. So the same thing happens to actions with long-term value in their pages. Once completed, they sink out of sight.

So as an action approach is done, pause for a moment and ask, is there any long-term value inside this page? Any insights, notes, instructions, decisions, emerging checklist or SOP style steps in the action page, you need to extract and preserve them so they don't become sediment.

## Step 2: Elevate

So how to prevent sediment and preserve insight? And now we come to the second part that's elevate. Elevate is about marking your best items with the no TV approach. And this approach works for both actions and neurobbits.

So note TV stands for notable topic and value.

### Notable

We'll start with notable. The notable button it's featured here in the C1 and C2 view. And we've talked about it. This button use it to mark that something is significant. Clicking it send this action to remembrance. It also I didn't mention it also send an action to your weekly and dual cycle highlights in addition to sending them to remembrance.

So with a simple click you'll rescue a memorable or meaningful item from Oblivion. It's the quickest and simplest way to preserve an item. This is why it comes first in the no TV approach in elevating an action. And this is why notable is in C1 and C2 because it's so easy to click a button.

### Topic Tagging

Now we go to topic tagging. When an action is done but certain durable insights is inside the page, you want to tag a topic. So it links the topic to the action itself. This makes sure that it also shows up in your knowledge space under the topic as a reusable source.

So for instance, update zoom link. This is tagging the topic SOP. So you will be able to find this action with rich content inside your topics.

### Value

So the last thing is value. Value is something that you want to use for sorting. So you want to reserve, we talked about reserve the highest values for items that are truly meaningful. So you don't want to inflate the the sorting with rating everything as very good or rating everything as excellent.

So with no and T alone, with notable and topic alone, an action is already very well preserved in other system contexts. If you don't have time to rate values in the moment, you can rate it later. During topic review or cycle review, you can scan these elevated actions and rate their values relative to each other.

So think of value as sorting. It's often the final step that decide what rises to the top. So you always start with notable and then topic. And if you don't have time, you can skip value for now because these two buttons already elevated the right items to the right place. Then you can rate them later.
