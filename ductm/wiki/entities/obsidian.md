# Obsidian

Ứng dụng quản lý ghi chú dạng markdown với hệ thống wikilinks và graph view. Được dùng như "IDE" cho LLM Wiki.

## Vai trò trong LLM Wiki

Theo [[andrej-karpathy]]: *"Obsidian is the IDE; the LLM is the programmer; the wiki is the codebase."*

- **Graph View** — hiển thị hình dạng wiki: page nào là hub, page nào orphan
- **Web Clipper** — browser extension clip bài viết web thành markdown → raw sources
- **Marp plugin** — tạo slide deck từ markdown
- **Dataview plugin** — query YAML frontmatter, tạo dynamic tables/lists
- **Download attachments** — tải ảnh về local để LLM có thể xem trực tiếp
