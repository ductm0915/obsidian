# Anthropic

Công ty AI tạo ra [[claude-code]] và dòng models Claude (Haiku, Sonnet, Opus). Được biết đến với triết lý "responsible AI."

---

## Products

- **Claude models**: Haiku (fastest/cheapest), Sonnet (balanced), Opus (heavy reasoning)
- **Claude Code**: Agentic coding tool — CLI + IDE extensions
- **Claude desktop app**: GUI wrapper cho Claude Code
- **claude.ai/code**: Web-based Claude Code chạy trên Anthropic cloud

---

## Notes

- Anthropic internally uses Claude Code for everything — built Claude co-work in ~10 days with 2–3 devs.
- AI đang được dùng để write và review tất cả code của Anthropic.
- Claude Code được tạo bởi Boris Churnney.
