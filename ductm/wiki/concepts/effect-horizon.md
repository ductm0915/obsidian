# Effect Horizon

**Origin:** [[sam-ovens]]
**Source:** [[sam-ovens-long-term-thinking]]
**Related:** [[second-order-consequences]], [[long-term-thinking]]

## The Framework

A visual decision-making tool: map any decision on a 2-axis chart.

- **X-axis:** Time (now → years out)
- **Y-axis:** Upside / Downside

Two options are almost always mirror images:

```
        Upside
          │        Option B (The Hill)
          │       /
          │      /
          │     /
──────────┼────────────────  Time →
          │     \
          │      \
          │       \
          │        Option A (Wheelchair Path)
        Downside
```

### Option A — The Wheelchair Path
- **Short-term:** Immediate upside, comfort, ease
- **Long-term:** Increasing downside as consequences compound
- Examples: Buying a TV on debt, cutting R&D for short-term profits, skipping workouts

### Option B — The Hill
- **Short-term:** Immediate downside, discomfort, effort
- **Long-term:** Increasing upside as positive feedback loops compound
- Examples: Reading instead of watching TV, investing in R&D, daily exercise

## The Rule

**Always take the hill.** The feedback loops at each decision node amplify the divergence exponentially, not linearly. A small initial difference compounds into a massive gap over time.

## Key Insight

The effect horizon reveals that most decisions are not between "good" and "bad" — they are between "good now, bad later" and "bad now, good later." The human brain is biologically wired for the first (dopamine, instant gratification), which is why this tool is necessary to override default instincts.
