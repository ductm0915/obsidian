# One Source Rule

## Definition
When learning a new skill or entering a new domain, **commit fully to one teacher/source for 90+ days** — consuming all their content, implementing everything, before moving to the next source.

## The Jigsaw Puzzle Analogy
Learning online is like solving a jigsaw puzzle. Most people:
1. Start a puzzle (source A)
2. Get to the hard part
3. Grab a second puzzle (source B) thinking it's easier
4. Dump both puzzles together
5. Now both are unsolvable

The fix: complete one puzzle fully, even when difficult — realizing there probably isn't an easier shortcut. Then move to the next puzzle.

## Why Multiple Sources Fail
- Different teachers contradict each other (each has a different complete system)
- Contradictions create paralysis ("which approach should I use?")
- The result: a head full of noise, notebook full of notes, no clear action
- You never go deep enough with any single approach to actually get results

## Application Rule
1. Find someone who has done what you want to do.
2. Commit: only consume their content until you've exhausted and implemented it all.
3. Achieve the results they promised.
4. Only then: move to the next source.

## When It Doesn't Apply
This rule is for **skill acquisition and implementation**, not for general knowledge expansion. Once you've achieved results in a domain, diversifying perspectives adds value. The rule applies *before* you have results, not after.
