# Vibe Coding for Beginners — Antigravity + Gemini + Claude Code

**Author:** Unknown instructor (manages business doing $4M+/year profit; teaches ~2,000 students)  
**Type:** Video course — vibe coding with Antigravity IDE + Gemini + Claude Code  
**Focus:** Building real products from simple portfolio sites to full SaaS apps  

## Overview

Beginner-to-advanced vibe coding course using the Antigravity IDE (Google-backed) as the primary environment, with Gemini 3.1 Pro as the main model and Claude Code as a secondary specialized tool.

## Core Toolkit

### Antigravity IDE
- Google-backed IDE (anti-gravity.google) designed for AI-native development
- Three-pane layout: file explorer (left) + editor (middle) + agent chat (right)
- Built-in Gemini integration with configurable workspace rules
- Supports Claude Code alongside Gemini
- Rule types: always-on / manual / glob-based
- Hotkeys: Cmd+B = toggle agent panel; Cmd+Shift+L = new conversation

### Gemini 3.1 Pro
Primary model for vibe coding. High-context, available to all Google account users. Used for: code generation, debugging, iteration, architectural decisions.

### Claude Code
Used alongside Gemini for specialized tasks. See [[claude-code]].

## What the Course Builds

1. Personal portfolio site (15 minutes)
2. Client-facing dashboard with auth + database
3. Lead generation SaaS (scrapes, enriches, sends emails)
4. AI thumbnail generator SaaS
5. Content automation SaaS (repurposes content across platforms)

## Key Concepts

### Vibe Coding
Working in natural language with AI to build software without needing to understand every line of code. The skill is: specifying what you want clearly, steering the AI when it veers off, and validating output.

### Workspace Rules
High-level instructions that guide model behavior. Function like CLAUDE.md in Claude Code — define style, constraints, and default behaviors.

### AI Overhang (same as [[agentic-workflow-course]])
Current AI capabilities far exceed what most people use. This course exists to close that gap for builders.

### Economics of Vibe Coding
Course covers: pricing, packaging, and selling vibe-coded products. Instructor explicitly addresses building for financial return, not just personal use.
