# qmd

Local search engine cho markdown files. Hybrid BM25/vector search với LLM re-ranking, chạy hoàn toàn on-device.

## Đặc điểm

- CLI interface (LLM shell out được)
- MCP server (LLM dùng như native tool)
- Hữu ích khi wiki phát triển lớn, vượt quá khả năng navigate bằng index.md

**GitHub:** https://github.com/tobi/qmd
